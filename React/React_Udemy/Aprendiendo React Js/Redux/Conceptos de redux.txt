		/// Conceptos de redux ///

	*/Actions*/

--Son objetos planos de javascript
	\\ tienen una key llamada type que indica el tipo de accion, puede contener otras propiedades como payload que sirven para ejecutar la accion
	\\ el type suele escribirse todo en mayusculas	
	\\ Los action Creators son funciones puras que devuelven actions. Son utiles para evitar inconsistencias en el codigo y tener que escribir los objetos a mano
		//ATENCION CON LOS ACTION CREATORS, QUE SIMPLEMENTE SON FUNCIONES QUE DEVUELVEN ACTIONS, Y ESTAN PRESENTES EN OTRAS DOCUMENTACIONES//
	
	*/ Reducers */

--Son funciones puras que reciben el estado anterior y la accion y a partir de ellas crea un nuevo estado de la aplicacion
-- Se les conoce tambien como generadores de estado

	*/ Store */

--almacena el estado global de la aplicacion. Permite que el estado sea leido, que te puedas subscribir a sus cambios y, lo mas importante, que envies acciones para crear un nuevo estado global
