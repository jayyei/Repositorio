////////////// Redux ///////////////////////
	
	Es un contenedor predecible del estado global de una aplicacion de javascript

	// Conceptos basicos //
	
	*El estado de tu aplicacion se describe como un simple objeto
	*El estado es global y puedes accederlo desde cualquier vista
	*El estado no se puede modificar, lo que haces es hacer un nuevo estado a partir del anterior mendiante acciones
	
	// Los 3 principios de redux //
	
	*Unica fuente de la verdad, esto es que solo hay un store en la aplicacion
	*El estado es de solo lectura, la unica forma de cambiarlo es emitir una accion que indique que cambio
	*Los cambios se realizan con funciones puras (acciones reducers)
